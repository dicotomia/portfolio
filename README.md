Este es el repositorio del portafolio profesional de **Ana Isabel Navarro Jiménez**, Desarrolladora Full Stack certificada en **AWS Cloud** e **IA Generativa**, especializada en la integración de sistemas, consumo de APIs y modelado de bases de datos relacionales, combinando un sólido bagaje técnico-sanitario con la arquitectura de software.

El sitio web está diseñado con un enfoque moderno, minimalista y responsivo (soporte completo para modo oscuro, animaciones sutiles y diseño fluido), sirviendo como carta de presentación para proyectos web, trayectoria laboral y certificaciones.

## 🚀 Proyectos Destacados

El portafolio expone proyectos clave desarrollados con diversas tecnologías:

1. **[Catfeína](https://github.com/ana-isabel-navarro/catfeina)**: Plataforma web integral para un catcafé con adopciones/apadrinamientos, reservas, e-commerce benéfico y pasarela de pago. Incluye gestión de usuarios con control de accesos por roles (adopciones, mesas, suscripciones, eventos) y despliegue contenerizado con Docker en Strato (Alemania).
   - *Stack:* Node.js, Express, MySQL, Tailwind CSS, Docker, JavaScript.
2. **[NutriSalud](https://github.com/ana-isabel-navarro/nutrisalud)**: Software clínico e-health para clínicas nutricionales con programación de citas, seguimiento antropométrico de pacientes e integración de APIs de alimentos.
   - *Stack:* PHP, Laravel, MySQL, Tailwind CSS.
3. **[DocuScan AI](https://github.com/ana-isabel-navarro/docuscan-ai)**: Herramienta en la nube de IA Generativa y Visión por Computador orientada al sector salud. Analiza, estructura y clasifica informes médicos e imágenes radiológicas bajo estándares HIPAA.
   - *Stack:* Python, Flask, AWS Cloud, Generative AI.

## 🛠️ Desarrollo Local

Este proyecto utiliza **Vite** para ofrecer una experiencia de desarrollo rápida.

### Prerrequisitos

Asegúrate de tener instalado [Node.js](https://nodejs.org/) (versión 18 o superior recomendada).

### Pasos para ejecutar localmente

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/ana-isabel-navarro/portfolio.git
   cd portfolio
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```
   Abre [http://localhost:5173](http://localhost:5173) en tu navegador para ver la web.

4. **Compilar para producción:**
   ```bash
   npm run build
   ```
   Esto generará la carpeta `dist` con los archivos optimizados listos para su distribución.
